# Conheça o Mendigo Lenhador

Este é um mini-documentário interativo feito em **HTML, CSS e JavaScript**.

## Sobre
Alex Louzada, mais conhecido como **Mendigo Lenhador**, tem uma história de vida difícil. Este projeto mostra sua história de forma visual e interativa, com animações de machado, rato, chuva e um botão “achou comida”.

## Como usar
- Clique no machado 🪓 para girar.  
- Observe o rato 🐀 andando e a chuva caindo.  
- Clique no botão “Clique aqui” para ver “🥖 Achou comida!”.

## Projeto
- HTML, CSS e JS puros  
- Foto hospedada via GitHub Pages  
- Layout adaptado para celular e computador
